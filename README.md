# Curso de MySQL - Curso em Vídeo
Esse repositório guarda todos os exercícios e testes que fiz seguindo as aulas.

## `O que eu aprendi:`
- [x] Compreender o que é um banco de dados, por que usamos, e como funciona na prática;
- [x] Instalar e configurar ambiente local usando ferramentas como o XAMPP para rodar MySQL no PC;
- [x] Criar o primeiro banco de dados e tabelas com comandos SQL básicos, usando CREATE DATABASE e CREATE TABLE;
- [x] Modificar a estrutura de tabelas existentes via comandos como ALTER TABLE, DROP TABLE, etc;
- [x] Inserir, atualizar e apagar dados usando comandos como INSERT INTO, UPDATE, DELETE, TRUNCATE;
- [x] Realizar consultas com o comando SELECT, incluindo filtros, agrupamentos, distinções de dados e funções de agregação para extrair informações úteis de bancos de dados;
- [x] Fazer backups e restaurações de bases de dados, compreendendo como exportar/importar dados, com segurança e organização.
